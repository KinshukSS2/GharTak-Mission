# 🕵️‍♂️ GharTak Mission: Bharatiya Missing Person Detection System  
*A humanitarian technology project to help bring missing loved ones back home.*

GharTak Mission (also known as **Bharatiya Rescue**) is a Django-powered missing person detection system that uses **facial recognition**, **real-time alerts**, and **secure location tracking** to assist families and authorities in reuniting missing individuals with their loved ones.

Built using **Django**, **OpenCV**, **face_recognition**, and a clean **HTML/CSS/JS** frontend, the system ensures a smooth user experience combined with strong backend reliability.

---

##  Features

###  Missing Person Registration  
- Register individuals with photographs and complete personal details.

###  Facial Recognition  
- Automated face detection and matching using OpenCV + `face_recognition`.

### Real-Time Alerts  
- Sends email notifications to parents/guardians and authorities when a match is found.

###  Location Tracking  
- Stores last known locations in a secure database.

###  Admin Dashboard  
- Admins can manage, update, and close missing-person cases.

---

##  How It Works

1. **Register a Missing Person**  
   Upload a photo and provide essential details like name, address, Aadhar number, email, and phone number.

2. **Surveillance Mode**  
   The system continuously scans faces, and if a match is detected, an alert email is automatically sent.

3. **Location Logging**  
   Each detection stores the last known location securely.

4. **Admin Management**  
   Admins maintain, update, or remove entries when a person is found.
